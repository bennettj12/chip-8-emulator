### Chip-8 Emulator Requirements
---
A chip-8 contains the following components:
* 16 8-bit registers
* 4k bytes of memory
* 16-bit index register
* 16-bit program counter
* 16-level stack
* 8-bit stack pointer
* 8-bit delay timer
* 8-bit sound timer
* 16 input keys
* 64x32 monochrome display memory
